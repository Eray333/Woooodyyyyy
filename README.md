# Woooody

## Theme
**Woooody** is a forest / woodland themed exploration experience focused on environmental audio immersion and spatial sound design.

## Game Mechanics
- FPS character controller (WASD movement, mouse look)
- Environmental interaction through proximity-based sound zones
- Main Menu with Play / Exit UI

## Game Feel (Juiciness)
- Footstep sounds vary based on surface (grass, wood)
- Pitch randomization is used to avoid repetitive audio feedback
- Smooth fade-in / fade-out ambience when entering forest areas
